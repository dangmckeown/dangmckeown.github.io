Fantasy Draft League standings app

Background

I'm part of a 'Draft League', using the scores from fantasy.premierleague.com. Each participating manager takes it in turns to pick, and is only allowed a maximum of one player per Premier League club. Unlikely legends of this format include West Brom's centre forward and Burnley's goalkeeper.

Since the Premier League site discontinued its json API, this season's standings are now based on scraped data.

For each of the managers, the app selects their best-scoring viable team (eleven on the pitch, one goalie, at least three in the midfield and defence and two up front). It then calculates the latest league standings.

Prime functions app

This app can return a range of prime numbers, test if a number is prime, or determine its prime factors. No commercial or military applications as of yet.

  • Try the Prime Functions app (Heroku)
  • View code on GitHub
A quick primer
  • Prime numbers are only divisible by themselves and one
  • All non-prime numbers have prime factors eg 6 can be produced from 2 and 3 (2 x 3 = 6) and 12 can be produced from 2, 2 and 3 (2 x 2 x 3 = 12)
